2. Please run command on smarte_project and run command npm install once installation is done
3. once npm install is finished run command "node server.js" or if you have nodemon you can run "nodemon server.js"
4. go to smarte-infogenie folder open cmd and run npm install.
5. after installation run command `npm start`
